# 餐廳手機掃碼點餐系統｜繁體中文化

## 功能
- 顧客手機點餐頁：依分類瀏覽餐點、加入購物車、填寫備註、送出訂單
- 後台訂單看板：查看桌號、餐點、金額、狀態，可標記製作中/已完成
- 菜單管理：新增餐點、上傳餐點照片、刪除餐點、修改餐廳名稱
- 桌號 QR 頁：產生各桌點餐連結範例

## 使用方式
1. 解壓縮檔案。
2. 直接打開 `index.html?table=A01` 測試顧客點餐頁。
3. 打開 `admin.html` 查看訂單後台。
4. 打開 `menu.html` 管理菜單。
5. 打開 `qrcode.html` 取得各桌點餐連結。

## 備註
此版本為可展示與測試的前端版，資料儲存在瀏覽器 localStorage，餐點照片會以瀏覽器資料方式保存，適合展示測試。
若要正式營業使用，建議加上：後端資料庫、登入權限、廚房出單機、Line Pay/信用卡付款、雲端部署。
