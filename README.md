# 餐廳手機掃碼點餐系統｜商業版（繁體中文）

## 功能
- 顧客手機掃碼點餐：桌號、分類、餐點照片、購物車、客製化選項、備註
- 後台管理：餐廳設定、菜單新增/編輯/刪除、照片上傳、上下架、資料備份匯入匯出
- 廚房 KDS：新訂單、製作中、待送餐、已送餐、列印
- 櫃台結帳：依桌號合併結帳、付款方式、未結帳桌次
- 營業統計：今日營業額、已結帳數、未結帳數、熱銷排行
- 桌號 QR Code：依桌數批量產生，可列印

## 登入
後台、廚房、櫃台、統計預設密碼：1234
請到「後台管理」修改密碼。

## 上傳 GitHub Pages
1. 建立 GitHub Repository。
2. 將本資料夾所有檔案上傳。
3. Settings → Pages → Deploy from branch → main → /root。
4. 完成後開啟 `index.html`。

## 注意
此版本為「免伺服器前端版」，資料儲存在同一台裝置的瀏覽器 localStorage。適合展示、測試、小型店內單機使用。
若要多台設備即時同步，需要加 Firebase / Supabase / 自架資料庫版本。
